# claw3d_analysis

`claw3d_test` 프로젝트를 건드리지 않고 별도로 분석 자료를 쌓기 위한 작업 폴더다.

## 목적

- 구조 분석 문서 누적
- 통합 설계 초안 정리
- 의사결정 기록 보관

## 원칙

- 이 폴더는 분석/설계 자산만 다룬다.
- 실제 앱 수정은 `../claw3d_test`에서 진행한다.
- 사실, 가설, 제안을 문서에서 구분한다.

## 구성

- `00_meta`: 분석 범위, 용어, 질문 백로그
- `01_system`: 시스템 구조와 실행 흐름
- `02_code_map`: 코드 위치와 책임 맵
- `03_integration`: 외부 서비스와의 통합 방향
- `04_product_design`: 제품/운영 관점 설계
- `05_decisions`: ADR 형식 결정 기록
- `06_gaps`: 한계, 리스크, 보완 포인트
- `07_plans`: 단계별 실행 계획
- `assets`: 다이어그램, 캡처 등 보조 자료

## 상태

현재는 문서 뼈대만 생성된 상태다.
