# Project Guide Feasibility Summary

## Project

- working name: `DeskMate`

## Bottom Line

- 구현 가능하다.
- 하지만 현재 기획 범위는 MVP로 바로 들어가기에는 넓다.

## Best MVP Shape

- assistant 1개
- agent 2개
  - Scheduler / Reminder
  - Wellness 또는 Code Review 중 1개
- 웹 대시보드
- 단순 IoT 알림

## Defer First

- 모바일 앱
- 삼성 헬스 연동
- 복잡한 multi-agent 협업
- 복잡한 approval / memory

## Main Risk

- 기능 욕심 때문에 구조보다 범위가 먼저 커질 가능성
