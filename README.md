# Ai_AGENTS_105_Test

개인 분석, 학습, 설계, 프로젝트 작업을 모아두는 허브다.

이 저장소의 기본 목적은 두 가지다.

- 외부 AI agent 오픈소스 저장소를 가져와서 분석하고 공부한다.
- 그 분석을 바탕으로 자체 개발 AI agent 프로젝트를 정리하고 키운다.

## Main Zones

- `forks`
  외부 저장소 체크아웃 위치다.
  예: `forks/claw3d`, `forks/openclaw`, `forks/hermes`
- `analysis`
  요청별 분석 메모와 소스별 분석 메모를 쌓는 곳이다.
- `knowledge`
  단어, 개념, 용어 설명을 쌓는 곳이다.
- `projects`
  자체 개발 프로젝트별 작업 공간이다.

## Current Default Workflow

1. 외부 저장소를 `forks/<repo-name>` 아래에 clone or fork checkout 한다.
2. 분석 결과는 `analysis` 아래에 누적한다.
3. 단어/개념 질문은 `knowledge` 아래에 정리한다.
4. 우리 프로젝트 관련 설계와 메모는 `projects/<project-name>` 아래에 정리한다.

## Boundaries

- 외부 소스 코드는 `forks`에 두고, 분석 문서는 `analysis`에 둔다.
- 외부 저장소 원문을 문서 폴더에 길게 복사하지 않는다.
- 용어 질문은 채팅으로만 끝내지 말고 `knowledge`에도 누적한다.
