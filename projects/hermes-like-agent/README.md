# Hermes-Like Agent

자체 개발 AI agent 프로젝트의 시작 폴더다.

현재 가정:

- Hermes 방식이 가장 가까운 참고 모델이다.
- OpenClaw와 Claw3D도 구조 비교 대상으로 본다.

## Subfolders

- `docs`
  프로젝트 핵심 문서
- `research`
  외부 참고 정리
- `decisions`
  결정 기록
- `notes`
  작업 중 메모

## Current Focus

- 어떤 runtime/orchestrator 구조를 채택할지 정리
- Hermes에서 가져올 것과 직접 만들 것을 나누기
- OpenClaw, Claw3D와의 경계 정리
