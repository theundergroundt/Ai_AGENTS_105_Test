# Presentation

발표자료 정리용 폴더다.

## Purpose

- 발표 메시지를 기술 분석 문서와 분리해서 관리한다.
- 슬라이드 초안, 비교표, 데모 스토리라인을 여기에 모은다.
- 분석 문서의 결론만 발표용 언어로 압축한다.

## Suggested Files

- `presentation-outline.md`
- `competitive-positioning.md`
- `demo-storyline.md`
- `notes/`

## Rule

- 발표 문서는 길게 설명하지 말고 메시지를 압축한다.
- 기술 세부사항은 `analysis/`에 두고, 발표 문서는 결론 중심으로 작성한다.
- 슬라이드 한 장에 들어갈 수 있는 수준으로 문장을 짧게 유지한다.
