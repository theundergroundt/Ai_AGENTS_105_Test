# 주석 분석 TS

이 폴더는 원본 `.ts` 파일을 직접 수정하지 않고, 복사본에 학습용 주석을 추가하는 공간이다.

## 원칙

- 원본 파일명 흐름을 유지하되 `.annotated.ts` 복사본으로 만든다.
- 실행용 코드라기보다 학습용 해설 코드로 본다.
- 설명은 "무슨 코드인가"보다 "왜 여기서 이 값을 만들고 다음 단계로 넘기는가"에 집중한다.

## 진행 순서

1. `01-server.impl.annotated.ts`
2. `02-server-runtime-state.annotated.ts`
3. `03-server-http.annotated.ts`
4. `04-ws-connection.annotated.ts`

## 현재 상태

- `01-server.impl.annotated.ts` 작성 완료
- `02-server-runtime-state.annotated.ts` 작성 완료
- `03-server-http.annotated.ts` 작성 완료
- `04-ws-connection.annotated.ts` 작성 완료
