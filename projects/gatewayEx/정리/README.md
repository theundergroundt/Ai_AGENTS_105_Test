# gatewayEx 정리 폴더

이 폴더는 `gatewayEx` 학습 내용을 정리하는 전용 공간이다.

현재 `gatewayEx`는 실행 가능한 완전한 프로젝트라기보다, OpenClaw gateway 코드 일부를 학습용으로 분리해 둔 구조에 가깝다. 원본 의존성은 상위 경로의 많은 모듈을 참조하므로 이 폴더만으로 바로 실행되지는 않는다.

## 정리 기준

- `gatewayEx` 관련 학습 메모는 이 `정리` 폴더 안에만 추가한다.
- 원본 `.ts` 파일은 학습 대상이고, 요약/해석/질문은 Markdown으로 남긴다.
- 새 메모는 목적에 맞는 하위 폴더에 추가한다.

## 현재 설계

- `01-구조/`: 폴더 구조와 파일 배치 파악
- `02-흐름/`: 서버 시작, HTTP, WebSocket 흐름 정리
- `03-파일/`: 파일별 역할과 핵심 포인트 정리

## 시작 순서

1. `01-구조/gatewayEx-구조.md`
2. `02-흐름/gateway-전체-흐름.md`
3. `03-파일/*.md`
